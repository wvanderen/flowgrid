# Sketch Manifest

## Design Direction

A sacred-geometry hex lattice that *breathes*. The Flowgrid canvas is always visible and never covered — tapping a module hex interacts with it **inline**, in place, no panel. Modules are sigil-marked patch nodes (Bespoke Synth lineage) with real Current flowing between them as glowing routes; the whole surface reacts to effort like a VJ rig. The mood is holographic, terminal-precise, electric/plasma, and a little alien — like peering into a vast, primordial dataspace (Ghost in the Shell's net) and waking it up. Feedback is dopaminergic and game-like: activating a module should feel like channeling plasma through a math lattice and watching it come alive.

Inline interaction is the frontier: how does using a module *on the grid itself* (never covered) actually look and feel? Sketches 001 (interaction paradigm) and 002 (scene/depth substrate) define the world; 003 (session view) and 004 (Current routing) fill in behavior.

## Reference Points

- **VJ visualizations** — signal-reactive glow, generative flow, light that responds to effort
- **Ghost in the Shell (the net)** — holographic depth, layered translucent planes, vast dataspace you dive into
- **Moonsigil Atlas** — sacred-geometry precision, deep cosmic palette, sigil glyphs
- **Super Hexagon** — the hex lattice as a pulsing, intense, hard-edge protagonist
- **Bespoke Synth** — modules as patchable nodes with visible signal flow between ports
