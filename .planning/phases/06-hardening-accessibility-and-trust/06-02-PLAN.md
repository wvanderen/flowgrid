## Phase Goal

**As a** Flowgrid user, **I want to** see my effort become visible motion — Current flowing, Bloom bursting, Core rippling — and trust that visuals never corrupt my real progress, **so that** the app feels alive and safe.

## Objective

Deliver the animated renderer vertical slice (UI-03 + UI-04): full-motion Pixi v8 visuals (D-01 particle trails/bursts/ripples, D-02 live ambient Current trail during active focus, D-04 forge/token visual events), the build-once/in-place-tween adapter refactor (D-05) so particle systems survive dispatches, the static-scene fallback when reduced motion is on or WebGL unavailable (D-03/D-08), the friendly WebGL-failure message (D-07), and the formal UI-04 safety property (dropping/skipping visual events never changes durable state). After this plan, the Flowgrid canvas is alive during and after focus, degrades gracefully, and is provably safe.

Purpose: UI-03 (full-motion visuals) and UI-04 (visual-event safety) are the heart of Phase 6's "alive + trustworthy" promise. D-05 is a mandatory structural prerequisite for D-01/D-02 (Pitfall 3: rebuild-on-dispatch kills particle systems). This plan depends on 06-01 for the `reduceMotion` field and `result.ts` is shared.

Output: Rewritten scene.ts (build-once + tween), new particles.ts/motion.ts/scene-inspect.ts, 3 new visual-event constructors + names, D-04 emissions in run-forge.ts/log-rejuvenation.ts, FlowgridCanvas.tsx rewire (update-in-place + reduceMotion + WebGL-fail message), UI-04 property test.

| Boundary | Description |
|----------|-------------|
| simulation → render | Transient visual events cross here (the renderer consumes but must not write back) |
| WebGL/GPU → canvas | Hardware-accelerated rendering may fail (old browser, headless, no GPU) |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-06-07 | Tampering | Particle system resource exhaustion | accept | Particle counts are small (bursts/trails); ParticleContainer is GPU-batched; reduced-motion fully stops the ticker; no unbounded growth (dead particles removed) |
| T-06-SC | Tampering | npm installs (none this plan) | accept | No new packages installed this plan — pixi.js 8.19.0 already present; @pixi/tween and @pixi/particle-emitter explicitly NOT installed (RESEARCH verified SLOP/incompatible) |
